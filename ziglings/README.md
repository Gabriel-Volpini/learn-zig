# Ziglings

A series of tiny broken programs to fix, designed to teach Zig syntax through practice.

## What is Ziglings?

Ziglings is a learning project that contains small exercises, each being a tiny broken Zig program. Your goal is to fix them and learn Zig fundamentals in the process.

## Getting Started

1. Make sure you have Zig installed
2. Explore the exercises in this directory
3. Fix each broken program to progress

## Exercises

The exercises are numbered and progressively teach different aspects of Zig:

- Exercise 1: Hello World
- Exercise 2: Variables
- Exercise 3: Functions
- ...and more!

## Running an Exercise

```bash
zig run exercise_01_hello.zig
```
